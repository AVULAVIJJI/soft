# Softmaster Mobile Apps

Mobile applications for the Softmaster platform.

## Apps

- **student-app** — For academy students (React Native)
- **employee-app** — For employees (React Native)

## Tech Stack

- React Native (Expo)
- TypeScript
- React Query
- Zustand

## Getting Started

```bash
npm install -g expo-cli
cd student-app
npx create-expo-app . --template blank-typescript
npm install
npx expo start
```

## API

All apps connect to: https://api.softmastertech.com
